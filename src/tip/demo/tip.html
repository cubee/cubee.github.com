API

    说明：
        信息提示、简单易用，适用于各种状态提示。
    使用：

        Y.Tip(id,status,msg,{config});
    参数：
        id:{string} 消息提示的容器
        status:{string} 此项为空时默认为tips，使用预设的类型 error、alert、attention、tips、ok、question、notice(s)、help(s)、small-help(s)、stop(s) *注意：(s)只有小提示才具有
        msg:{string} 消息主体
	配置：
        bigtip:{boolean} 默认是小提示;true:24x24图标大提示
        anim:{boolean} 默认有动画效果;
        delay:{number} 默认是3秒
        autohide:{boolean} 默认不隐藏；true:自动隐藏
        noborder:{boolean} 默认有边框；true:无边框
        onfinish:{function} 提示隐藏后的回调

以下为所有消息原型

16x16

错误信息error，比如：请正确选择省市区

禁止信息stop，比如：对不起，你是小二，不能出价

警示信息alert，比如：淘宝网12月25日凌晨1:00 - 3:00将停机维护

提醒信息attention，比如：建议您联系卖家咨询货物是否已经发出

提示信息tips，比如：您可以在该类目下直接发宝贝哦

成功信息ok，比如：交易成功

公告信息notice，比如：你有3条信息未读

疑问question，比如：什么是宝贝成色

可以在class中加入naked, 去除边框和背景色

帮助信息help，比如：查看帮助

12x12版的帮助信息help，比如：查看帮助
